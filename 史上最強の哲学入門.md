## 本の目次とあらすじ

- 各哲学者が某バキの格闘前の実況風に紹介されている 計32人
  - 当初、普通の哲学者の紹介をしたかったが、何かが足りないと考えた結果「バキ成分が足りない」と結論ついた模様
  - バキ -> 最強の男を目指す
  - 哲学者 -> 最強「論」を目指す(誰もが正しいと認めざるを得ない論)
  - 以下各人でまとめるが、大量すぎるので気になった人のみ。
  - 省略あり
- 1R 真理の「真理」
  - プロタゴラス
    - 『相対主義』
    - 「人間は万物の尺度である」
      - ぶっちゃけ真理なんて人それぞれだよ論
  - ソクラテス
    - 『無知の知』
      - 自分が「無知」だからこそ「真理」を追い求めようという論
      - 相対主義があるなら真理なんて探す必要なくない？論をブン殴るために手を挙げて作ったやつ
  - デカルト
    - 『方法的懐疑』
    - 「我思う、ゆえに我あり」
      - 全てが夢やウソだとしても、それらを疑っている自分がいることは確実であるという論
      - 絶対的に正しいといえる公理を定義し、そこから定理として発展させていく過程で誕生
  - ヒューム
    - 『懐疑論』
    - 複合概念
      - 神も化学も人間が過去に経験した物事が複合したもの、つまりは想像にすぎないという論
  - サルトル
    - 『アンガージュマン』
    - 「自由の刑」
      - 人は正解等与えられず、ただ何が正しいかも分からない状況で「好きにしろ」と放り出された状態と定義
  - デューイ
    - 『実用主義（プラグマティズム）』
    - 「道具主義」
      - 「～とは何か」等の問いかけは埒があかないから、「～は一体何に役に立つのか、その効果は何か」を考えようとした主義
  - デリダ
    - 『脱構築』
    - 「読み手中心主義」
      - 従来の読み手（話し手）が持つ意図（唯一絶対の解釈）を探るのではなく、話し手の話から聞き手が得た解釈が正解とすれば良いよねというやつ
      - どれだけ会話等をしても、「言葉」を介している限り「意図」は読み取れない
        - 解釈が正しいなんて判断不能なため、いっそのこと「聞き手」の解釈それぞれが正解にしようという考え
  - 結局の「真理」とは？
    - 最初の相対主義から、色々と神なら色んな主義が出て、最終的に道具主義というほぼ相対主義に似た考えに一周して戻ってきちゃった。主に理由は２つ
      1. 真理を求める闘争は致命的
          - 核やら戦争やら危なっかしいものがある恐ろしい時代に「これが絶対正しい！」なんぞ言えば即滅亡ルートになるので、相対主義にならざるをえない
      2. あらゆる学問での限界突破
          - 物理の「不確定性原理」
            - どんなに頑張っても観測できない領域が存在するよってやつ
            - 観測できなければ理解や解釈も出来ないので、探究できないよねっていう証明
          - 数学の「不完全性定理」
            - 究極の論理的体系は絶対に作れず、どんな体系でも必ず証明不可能な命題が発生してしまうという定理
      - つまり、考えや法則の外側 = 「他者」が存在する以上、絶対的な原理は存在しないことが学問的に証明されてしまった
  - レヴィナス
    - 『他者論』
      - どんな学問でも「その囲い」を否定する外側、「他者」が存在することを示した
      - 絶対的な真理は無いことを証明してしまった
      - 他者とは「意図」の確実な疎通が出来ない不愉快な存在であり、そのぞんざいがあるからこそ、「新しい価値観や真理」が無限に創造させられる
- 2R 国家の「真理」
  - プラトン
    - 『イデア論』
      - その物体を表す「究極の理想の存在」である『イデア』は人々の頭の中にあるよってやつ
      - 哲学も究極な存在であるイデアが存在するので、それを証明できれば真理になりえるよねと考えた
  - アリストテレス
    - 『論理学』
      - イデアあっても頭の中にある以上、証明できないし、あっても何の役にも立たないよねという考え
    - 3つの政治体制である君主制、貴族制、民主制は全て腐敗しやすいことも証明している
      - 君主制 -> 独裁制
        - いわずもがな
      - 貴族性 -> 寡頭制（かとう）
        - 寡頭制 : 貴族が権力争いしてぼろぼろになっていくさま
      - 民主制 -> 衆愚制
        - みんなで考えるので、他の人が勝手にやってくれるだろと無関心になって適当になってしまうやつ
  - ホッブス
    - 『社会契約説』
      - 国家とは皆が互いに殺しあわないように作った組織だよというやつ
      - そのために警察などの武力といった「恐怖」を用いて統治しているよとも説いている（所謂レヴァイアサン）
  - ルソー
    - 『人民主権』
      - 真の権力者は王ではなく民衆であるよと説いたやつ
      - 民衆がいるから王が成り立っているよという考えから、「フランス革命」の火種にもなった
  - アダム・スミス
    - 『見えざる手」
      - 皆自由にやっても最終帝に良い方向に見えざる手が誘導してくれるよ論
      - 富を優先しても最終的にめぐり回って、社会全体の利益に広がるような結果になると説いた
    - 経済学を生み出した父と呼ばれる人でもある
  - マルクス
    - 『共産主義』
      - 俗にいう「富を皆に分配しようよ」考え
      - まぁ色々問題が発生したのはソビエトとかの例を見れば言わずもがな
      - ただし、資本主義が持つ問題点等を明確化した点がある
  - 労働の価値
    - ニートなど、労働そのものの価値が最近薄れている傾向
    - だからと言って国家は絶対に「何もしません」
    - 安価な娯楽が大量にある今こそ、新しい主義が必要
